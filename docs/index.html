耿同学.skill / GengScope

面向公开论文的科研完整性索引工作流。中文展示名是耿同学.skill，英文包名、CLI 和 API 名称保持为 gengscope。GengScope 汇总公开状态、材料覆盖率和算法审计信号，但不判定论文造假。

Evidence-linked research integrity indexing workflow for public papers. The Chinese display name is 耿同学.skill; the English package, CLI and API name remains gengscope. GengScope reports public statuses, material coverage and algorithmic review signals, but does not determine misconduct.

撤稿校准 / Retraction Calibration

当前盲校准流程先让 GengScope 在不读取撤稿原因的情况下提取信号，再与官方撤稿原因对齐。最新 20 篇案例 gate 完成 20/20，其中 18/20 至少有一个盲提取信号族与官方原因匹配。

The current blind calibration run compares GengScope signals against official retraction reasons only after the blind pass. The latest 20-case gate completed 20/20 cases and matched 18/20 by at least one blind signal family.

新加的 5 篇案例 demo 展示了盲信号、证据文件/区域、官方原因和对齐状态。
