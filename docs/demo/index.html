撤稿校准 / Retraction Calibration

最新回顾性校准使用已撤稿论文作为标签，但只在盲信号提取完成后才读取官方撤稿原因。
The latest retrospective calibration run used already retracted papers as labels only after the blind signal pass.

流程会导入每篇已撤稿论文，发现可用材料，在不读取撤稿原因的情况下运行 numeric/image/metadata 信号提取，然后把信号族与官方撤稿通知对齐。它是改进复核召回率的校准证据，不是独立的 misconduct 分类器。

The workflow imports each retracted article, discovers available material, runs numeric/image/metadata signal extraction without reading the reason, then aligns signal families with the official retraction notice. It is calibration evidence for improving review recall, not an independent misconduct classifier.

The five-case alignment demo shows each paper's blind signal, evidence file/region pointer, official reason and alignment status.
