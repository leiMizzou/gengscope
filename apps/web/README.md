# Web App Plan

Target stack:

```text
Next.js
TypeScript
Tailwind CSS
TanStack Query
```

Initial pages:

```text
/                         Search-first home
/papers                   Paper search results
/papers/[doi]             Paper detail
/institutions/[id]        Institution detail
/review                   Review workbench
/api-console              Agent API console
```

First components:

```text
SearchBox
RiskBadge
RiskCard
EventTimeline
EvidencePointerList
AuthorInstitutionTable
InstitutionMetricsPanel
AlgorithmSignalTable
SourceArtifactViewer
```

UI direction:

- Operational and dense.
- Table-first.
- Strong source links.
- No dramatic wording.
- No institution shame ranking.
